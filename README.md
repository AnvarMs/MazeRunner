## 👻 About

A **procedurally generated horror maze experience** for PC, Mobile, and WebGL — every playthrough generates a completely different maze, so no two runs are the same. Navigate endless shifting corridors while an intelligent AI enemy hunts you down.

Maze layouts are built using **recursive backtracking**, and the horror NPC hunts using **BFS-based pathfinding** across the generated maze graph. A custom cross-platform input system resolves touch/mouse conflicts to keep a stable 60 FPS on both mobile and WebGL builds.

## ✨ Current Features

- 🌀 **Procedurally generated maze layouts** with dynamic pathfinding — recursive backtracking generation, different every run
- 🤖 **Intelligent horror NPC** with patrol, chase, and catch states, using BFS pathfinding to hunt across the maze
- 🎮 **Cross-platform first-person controls** — desktop and mobile, built on Unity's New Input System
- 🔊 **Atmospheric positional audio system**
- 🖱️ **Smooth camera controls** with customizable sensitivity

<br/>

## 🕹️ How to Play

**Desktop**
- `WASD` / Arrow Keys — move
- Mouse — look around
- Sensitivity adjustable in the settings menu

**Mobile / Touch**
- Virtual joystick — movement
- Swipe anywhere — look around
- Touch controls auto-detected

**Objective:** find your way through the ever-changing maze while avoiding the horror NPC. It patrols, and once it spots you, the chase begins — get caught, and you're teleported back to the start.
